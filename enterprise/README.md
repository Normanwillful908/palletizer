# Enterprise Extensions

This directory is reserved for enterprise-grade extensions to the Palletizer Full Stack. These modules are designed for organisations that need to go beyond the open-source foundation and require production-hardened, commercially supported capabilities.

## What's Coming

### Enterprise Connectors (`connectors/`)

Pre-built integrations for industrial automation platforms:

- **KUKA Sunrise** connector for KUKA robot arms
- **ABB RobotStudio** connector for ABB IRB series
- **Fanuc ROBOGUIDE** connector for Fanuc robots
- **Universal Robots RTDE** connector for UR cobots
- **Siemens TIA Portal** connector for PLC integration
- **Rockwell FactoryTalk** connector for Allen-Bradley systems

### Advanced Algorithms (`algos/`)

Production-optimised algorithms for high-throughput cells:

- **Multi-SKU layer optimisation** — mixed-case palletising with stability constraints
- **Dynamic path planning** — collision-free trajectories in shared workspaces
- **Predictive maintenance** — ML-based fault prediction from vibration and thermal data
- **Energy-optimal scheduling** — minimise power consumption while meeting throughput targets

### Why Enterprise?

The open-source stack gives you everything you need to build, test and deploy a palletiser. The enterprise extensions are for teams that need:

- Certified connectors with guaranteed compatibility and SLA-backed support
- Algorithms validated against real-world throughput benchmarks
- Priority access to new features and direct engineering support
