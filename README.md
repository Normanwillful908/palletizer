## 🧰 Troubleshooting Tips

Here are common issues you might face and simple fixes:

- **Robot does not connect:** Check cables and power, verify correct port or IP.  
- **Pallet layout looks wrong:** Review product size and pallet type settings.  
- **App runs slowly:** Close other programs and make sure your PC meets system requirements.  
- **Installation failed:** Run the installer as an administrator and retry.

If problems continue, visit the GitHub Issues page for palletizer to find help or submit a bug report.
